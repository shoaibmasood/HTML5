    Project:

    Vehicle Meter Testing Panel:

    In an automotive industry of Pakistan, majority of the companies uses manual testing systems for their produced meter panels that require an operator to check whether the meter panel working perfectly or not. Therefore due to human intervention probability of error occurrence is maximum. Therefore keeping this issue in our mind we came up with an idea of automatic vehicle meter panel testing machine which is aimed to inspect and test vehicle meter panels automatically. The Main objective of this project was to design an automated system that can inspect a vehicle meter and then forward it to the next process if it fits the requirement.
    • To interface an external camera with Raspberry Pi in to acquire images.
    • To detect faults/defects in the vehicle meter in order to reject it.
    • To operate the entire system without human intervention
    • To reduce testing time hence increase in production by using “Raspberry-Pi” module (Linux based) and implementing Image Processing Techniques.
